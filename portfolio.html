1.1.9 Build A Bear Project


Build your own bear! It adds a little bit more love to life. Our invention is exactly like the ones in real life, but better. Add some style with your bear’s necklace, hat, and phone. Make your bear any color with any accessory, with endless combinations! Once you start the code, you are presented with a series of questions within VS code. This series of questions include yes/no and color-based questions to help you personalize your bear. Depending on your answers to these questions, the code will execute a specific response to apply it on the bear.

1.2.5 Minion Stacker Project


The name of our game is called Minion Stacker. This game includes a timer for how long you have till the round is over. The game is designed to get points based on how many minions you stack before the round is finished. Some minions like the purple ones can get you more points sense they are more rare than the other ones. You are only able to move left and right but not up and down. Once you start the code, The timer starts as well. Minions slowly but surely fall from the sky. The minions are randomized and the randomizer only includes the purple minion one out of 10 minions. Once you see a Minion then you are able to move them by using the A and D ( left and right). Once you are able to move the minions you can start placing them on eachother. Keep in mind that the timer is short so you have to have the skills to be able to score high.

Scratch: Racing Minions


The name of our game is called Racing Minions. In this game, you race against a CPU aka “The Evil Minion” and you need to cross the finish line first in order to win. The game is designed in a way that you first choose a one of the three minions given, then you race against the CPU for 3 laps. If you win, the minion you chose will appear on the win screen. However, if you lose, the Evil Minion will appear on the lose screen. Once you start the code, you will have to choose one of the three minions given. After that, a countdown will start and soon enough you will start racing against the CPU for 3 laps. To control your car, you use the arrow keys (right key to move right, up key to accelerate, left key to move left, etc.). If you finish first, then you are presented with a Win screen with some positive music with you chosen minion. However, if you lose, you are presented with a Lose screen with some negative music with the evil minion.
